# Botanical Ingredients Frame Sequence

## Instructions

Place your frame sequence images in this folder with the following naming convention:

```
frame-0001.jpg
frame-0002.jpg
frame-0003.jpg
...
frame-0060.jpg
```

## Requirements

- **Total frames**: 60 images
- **Format**: JPG (recommended for performance)
- **Naming**: `frame-XXXX.jpg` where XXXX is a 4-digit number (padded with zeros)
- **Content**: Images should show the Lumina Botanical jar with eucalyptus, peonies, and water droplets
- **Color scheme**: Soft greens, whites, and natural tones
- **Recommended resolution**: 1920x1080 or similar HD resolution

## Frame Distribution

- **Frames 1-20**: Focus on Eucalyptus leaves
- **Frames 21-40**: Transition to Peonies
- **Frames 41-60**: Highlight water droplets and hydration

## Tips

- Ensure smooth transitions between frames for seamless scrolling
- Keep the jar centered in all frames
- Use consistent lighting and color grading
- Export at high quality but optimize file size for web performance
